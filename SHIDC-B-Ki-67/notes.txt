File Naming: 
File names follow the "PATIENT_IMGAGE_CROP" rule. 
Image and it's corresponding label file has the same name.

For example:  "p1_0299_6.json" and "p1_0299_6.jpg"

Label:
Label files have labeled nuclei coordinates and the label id ( 1: Positive Stained, 2: Negative Stained, 3: TIL)
